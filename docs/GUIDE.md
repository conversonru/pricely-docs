## Что такое Pricely

**Pricely** — сервис для оптовых компаний России. Клиент загружает Excel-прайс и за 30 минут получает:

1. **SEO-сайт-каталог** — индексируется Яндексом и Google, у каждого товара своя страница
2. **Telegram Mini App** — каталог прямо внутри Telegram

**Целевая аудитория:** оптовые поставщики (стройматериалы, автозапчасти, одежда, промышленность).

## Как работает SEO-каталог

### Адреса страниц

| Страница | URL | Описание |
|----------|-----|----------|
| Список товаров | `/catalog/demo` | Все товары с поиском и фильтрами |
| Карточка товара | `/catalog/demo/vtulka-rezinovaya-50h30` | Один товар с кнопкой заказа |

### SEO

Каждая страница генерируется **на сервере** (SSR) — Яндекс и Google видят полный HTML со всеми товарами, ценами и описаниями без запуска JavaScript.

Карточка товара содержит **Schema.org разметку** — позволяет Яндексу показывать товар с ценой прямо в поисковой выдаче (rich snippet).

### Кнопка заказа

При нажатии на "Заказать в WhatsApp" или "Заказать в Telegram" открывается переписка с готовым сообщением:
> _"Здравствуйте! Хочу заказать: Втулка резиновая 50х30 (арт. R-001)"_

Клиенту сервиса не нужно ничего настраивать — система автоматически подставляет название товара и артикул.

## Как работает Telegram Mini App

### Адреса страниц

| Страница | URL | Описание |
|----------|-----|----------|
| Список товаров | `/tma/demo` | Каталог в стиле Telegram |
| Карточка товара | `/tma/demo/vtulka-rezinovaya-50h30` | Товар с кнопкой заказа через Telegram |

### Особенности TMA

- Страницы используют **цвета темы Telegram** — светлую и тёмную, как у пользователя в настройках
- **Нативная кнопка «Назад»** в шапке Telegram (не в браузере) — возвращает на список товаров
- Кнопка заказа открывает чат с поставщиком напрямую внутри Telegram
- TMA-страницы **не индексируются Яндексом** — это нормально, они для Telegram

### Как открыть TMA в Telegram

1. Создать бота через `@BotFather` → `/newbot`
2. Подключить Web App: `/newapp` → указать URL `https://pricely.ru/tma/demo`
3. Пользователи открывают каталог кнопкой внутри Telegram

> В браузере TMA открывается как обычная страница (без Telegram-функций — это нормально для разработки).

## Переменные окружения

Файл `.env.local` (на твоём компьютере, не публикуется в git):

```env
# Адрес Supabase на сервере
NEXT_PUBLIC_SUPABASE_URL=http://72.56.5.159:8000

# Публичный ключ (anon) — для чтения данных в каталоге и TMA
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJ...

# Секретный ключ (service_role) — для записи данных из Make.com
SUPABASE_SERVICE_ROLE_KEY=eyJ...

# Telegram Bot Token (для настройки Mini App)
TELEGRAM_BOT_TOKEN=

# Секрет для webhook Make.com (генерируется в День 5)
UPLOAD_SECRET=

# Адрес приложения (меняется при деплое на Vercel)
NEXT_PUBLIC_APP_URL=http://localhost:3000

# Домен сервиса
NEXT_PUBLIC_APP_DOMAIN=pricely.ru
```

> ⚠️ Никогда не публикуй `SERVICE_ROLE_KEY` в открытом доступе — он даёт полный доступ к базе данных.

## Часто задаваемые вопросы

**Q: Как добавить нового клиента?**  
Пока вручную через Supabase Studio (Table Editor → clients → Insert row). В День 5 будет автоматизация через Make.com.

**Q: Как добавить товары?**  
Через n8n: редактируешь Google Sheets, n8n запускает синхронизацию → товары обновляются автоматически. Или вручную через Table Editor → products.

**Q: Почему n8n, а не Make.com?**  
Make.com — облачный сервис, данные проходят через серверы в Европе/США — нарушение 152-ФЗ. n8n self-hosted на московском VPS: все данные остаются в России, и стоимость $0 (VPS уже оплачен).

**Q: Почему Supabase self-hosted, а не облачный?**  
Облачный Supabase хранит данные во Франкфурте — нарушение 152-ФЗ. Self-hosted на Timeweb Moscow = данные в России.

**Q: Можно ли дать клиенту свой домен?**  
Да, это планируется в День 6. Клиент сможет использовать `catalog.mycompany.ru` вместо `mycompany.pricely.ru`.

**Q: Как Telegram Mini App отличается от обычного сайта-каталога?**  
TMA открывается внутри Telegram как нативная панель, использует тему Telegram (светлую/тёмную), имеет нативную кнопку «Назад» и кнопки заказа, которые открывают чат напрямую в Telegram. Обычный каталог — для SEO и внешних пользователей.

**Q: Почему `/tma/` страницы не индексируются Яндексом?**  
TMA — Client Components (код работает в браузере), Яндекс-бот не выполняет JavaScript. Это нормально: TMA нужен для Telegram-трафика, а не для поискового.
